Key Observations
================
1. Float32 achieves 83.69% accuracy with excellent silence/unknown recognition
   - Silence F1: 0.949 (can reliably detect pauses)
   - Unknown F1: 0.809 (can reliably reject out-of-vocabulary speech)
   - These were unmeasurable before (test set had zero silence/unknown samples)

2. INT8 quantization loss is acceptable at 3.64 percentage points
   - Worst-affected class: tala (F1: 0.671 → 0.580, Δ -0.091)
   - Best preserved: baalnu (F1: 0.872, Δ 0.000)
   - Silence still performs well (F1: 0.920)

3. Per-keyword performance patterns:
   - Strongest: feri (0.947 float32), maathi (0.933)
   - Weakest: roknu (0.663 float32), tala (0.671)
   - These are phonetically confusable pairs in Nepali

4. The 12 keywords average 84.6% F1 (float32), 83.0% F1 (int8)
   - With silence+unknown context, overall robustness is 83.69% / 80.05%

Test Set Construction Details
==============================
To build a proper 14-class evaluation set:
1. Randomly sampled 50 raw silence clips from data/raw/_silence/silence/
2. Moved to data/raw/_silence_test/silence/ (no data leakage to train)
3. Moved corresponding processed features (.npy) to data/processed/_silence_test/silence/
4. Repeated for unknown: 50 raw clips → data/raw/_unknown_test/unknown/
5. Updated configs/speaker_split_v1.json to include _silence_test and _unknown_test in test split
6. Verified loader recognizes all 14 classes: KEYWORDS list already included silence/unknown

Final test composition verified:
  - 1183 total samples
  - 1133 keyword samples (90 per keyword × 12 + 90 + 93)
  - 50 silence samples
  - 50 unknown samples
  - 5 pseudo-speakers (voicer28, voicer29, voicer30, _silence_test, _unknown_test)
